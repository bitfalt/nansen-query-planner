# Demo Runbook

## Goal
Show Thesis Battlefield as a practical intelligence layer on top of Nansen CLI output.

## Demo sequence
1. State the thesis.
2. Show the planned query sequence.
3. Run plan mode or bounded execute mode.
4. Open the generated markdown report for the human view.
5. Open the generated JSON report for the agent/LLM-consumable view.
6. Explain the verdict, confidence, and next questions.

## Important line to say
This is not just another raw query wrapper. It is an intelligence layer that turns Nansen data into a structured, reusable verdict for both humans and agents.
